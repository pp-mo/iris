* The ability to pass file format object lists into the rules based load pipeline, as used for GRIB, Fields Files and PP has been added.  The :func:`iris.fileformats.pp.as_cubes`, :func:`iris.fileformats.grib.as_cubes`, :func:`iris.fileformats.pp.as_load_pairs` and :func:`iris.fileformats.grib.as_load_pairs` are provided to produce cubes from such lists.  These lists may have been filtered or altered using the :mod:`iris.fileformats` modules.
